# ADH Hardware Store

A full-stack hardware and construction materials shopping website.

## Features

- User Registration & Login
- Product Catalog
- Search Products
- Shopping Cart
- Checkout
- Order History
- Admin Dashboard
- Product Management
- Order Management

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
